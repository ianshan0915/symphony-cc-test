# Content Writing

## When to use this skill

Activate this skill when the user:
- Asks to write or draft any type of content
- Needs help with blog posts, articles, or documentation
- Wants to create emails, proposals, or reports
- Requests marketing copy or social media content

## Writing process

### 1. Brief analysis
Before writing, clarify:
- **Purpose**: What should this content achieve?
- **Audience**: Who will read it? What do they already know?
- **Format**: Blog post, email, report, documentation, etc.
- **Tone**: Professional, casual, technical, persuasive?
- **Length**: Approximate word count or page target
- **Constraints**: Brand guidelines, style preferences, deadlines

### 2. Structure planning
For content over 300 words, propose an outline first:
- Opening hook or executive summary
- Key sections with logical flow
- Supporting evidence or examples
- Clear conclusion with call to action (if appropriate)

### 3. Drafting
- Write in the appropriate voice and tone
- Use active voice by default
- Keep paragraphs focused (one idea per paragraph)
- Use transitions between sections
- Include specific details over vague generalities

### 4. Refinement
- Eliminate filler words and redundancy
- Verify all claims and data points
- Ensure consistent terminology throughout
- Check that the opening delivers on its promise

## Content type templates

### Blog post
See [references/blog-template.md](references/blog-template.md) for the standard blog post structure.

### Technical documentation
- Start with a one-sentence summary of what the feature/API does
- Include prerequisites and setup
- Provide working code examples
- Add troubleshooting section for common issues

### Email
- Subject line: specific and action-oriented
- Opening: context in one sentence
- Body: the key message or request
- Closing: clear next step or call to action

### Report
- Executive summary (findings + recommendation in 2-3 sentences)
- Methodology
- Findings with supporting data
- Recommendations
- Appendix for detailed data

## Style guidelines

- **Clarity**: If a simpler word works, use it
- **Concision**: Every sentence should earn its place
- **Specificity**: "Revenue grew 23% in Q3" beats "Revenue grew significantly"
- **Parallelism**: Keep list items in the same grammatical form
- **Scannability**: Use headings, bullets, and bold text for key points
