# Web Research Reference

## Search query optimization

- Use quotes for exact phrases: `"machine learning" applications`
- Use domain filters: `site:arxiv.org transformer architecture`
- Combine specific terms: prefer `Python asyncio timeout handler` over `Python async`
- Include date qualifiers for recent info: `GPT-4o benchmarks 2024`

## Source credibility hierarchy

1. **Primary sources** — official documentation, peer-reviewed papers, government data
2. **Established media** — major news outlets, industry publications
3. **Expert blogs** — recognized domain experts with verifiable credentials
4. **Community sources** — Stack Overflow, Reddit (useful but verify independently)
5. **Social media** — treat as leads, not sources

## Common pitfalls

- Confirmation bias: search for counter-evidence, not just supporting evidence
- Recency bias: newer is not always more accurate
- Authority bias: even reputable sources can be wrong on specific claims
